# Event Information Portal Demo

A public portfolio demo of a mobile-first event information portal.

This repository contains only generic sample data and demo SVG assets. It does not include real attendee data, real private event details, internal forms, private Google Sheet URLs, API keys, passwords, or identifiable photos.

## Demo Features

- Mobile-first event guide layout
- Accordion navigation
- Schedule and menu tabs powered by JSON files
- Demo announcements
- Demo team search with autocomplete
- Image preview modal
- Generic lodging, maps, transportation, and flyer assets

## Editing Demo Data

- Schedule: `data/schedule.json`
- Menu: `data/menu.json`
- Demo announcements and team search data: `script.js` and `preaching.js`

All content is intentionally fictional.
